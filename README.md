README - FRONTEND (React + TypeScript + Tailwind + Redux Toolkit)
==================================================================

📁 Dossier : project/

------------------------------------------------------------------
🔧 PRÉREQUIS :
- Node.js ≥ 16.x
- npm ou yarn

------------------------------------------------------------------
📦 INSTALLATION :
cd project
npm install
# ou
yarn install

------------------------------------------------------------------
🚀 DÉMARRER EN DÉVELOPPEMENT :
npm run dev
# ou
yarn dev

Ouvre http://localhost:3000 dans ton navigateur.

------------------------------------------------------------------
🏗️ BUILD POUR LA PRODUCTION :
npm run build
# ou
yarn build

Le résultat est dans le dossier /dist ou /build.

------------------------------------------------------------------
🧪 TESTS (si configurés) :
npm run test
# ou
yarn test

------------------------------------------------------------------
🔍 LINTER & FORMATAGE :
# Vérifier les erreurs de code
npm run lint
# ou
yarn lint

# Formater automatiquement (si Prettier est configuré)
npm run format
# ou
yarn format

------------------------------------------------------------------
🧼 NETTOYER LE PROJET (réinstaller proprement) :
rm -rf node_modules .cache dist
npm install
